## • Compilation Instructions: 
        1. Clone repository or extract all in zip folder.
        2. open project as a maven project.
        3. Check that project SDK is Amazon Coretto 23.
        4. For using IntelliJ’s Run button, add the following to the VM options:
        --module-path "<user home>/.m2/repository/org/openjfx/javafx-base/21.0.6/javafx-base-21.0.6-win.jar;... (other JavaFX jars) ..." --add-modules javafx.controls,javafx.fxml
        else, using maven, do mvn clean install javafx:run.
        5. Build and run the main class: `com.comp2042.Main`.
    The game should launch a javaFX window.
